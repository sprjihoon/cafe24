## 🎯 주요 기능

### A. 실제 쇼핑몰 모드 (기본 동작)

#### 1. 실제 리뷰 0개일 때
```
빈 상태 메시지 표시:
─────────────────────────────
아직 첫 번째 착용 후기를 기다리고 있습니다.
상품을 경험하신 뒤, 당신의 일상을 들려주세요.
─────────────────────────────
→ 화살표, 페이지네이션 숨김
→ 슬라이더 비활성화
```

#### 2. 실제 리뷰 1~2개일 때
```
→ 실제 리뷰 카드 표시
→ 화살표, 페이지네이션 숨김
→ 슬라이더 비활성화
```

#### 3. 실제 리뷰 3개 이상일 때
```
→ 실제 리뷰 카드 표시
→ 화살표, 페이지네이션 표시
→ 슬라이더 활성화
→ 드래그, 스와이프 가능
```

### B. 데모 모드 (디자인센터 전용)

**활성화 조건**: `ATELIER_DEMO_MODE = true`

```
실제 리뷰 0개 → 데모 리뷰 5개 표시
→ 슬라이더 활성화
→ 화살표, 페이지네이션 표시
→ 상품 정보 표시
```

**중요**: 실제 리뷰가 1개라도 있으면 데모 모드는 무시되고 실제 리뷰만 표시됩니다.

## 🎨 디자인 사양

### 카드 구조
```
┌─────────────────────┐
│   리뷰 이미지       │ ← 1:1 비율, object-fit:cover
│   (정사각형)        │
├─────────────────────┤
│ ★★★★★ (별점)      │
│ 리뷰 제목           │
│ 리뷰 본문 (2-3줄)  │
│ 작성자 • 날짜      │
├─────────────────────┤
│ [썸네일] 상품명     │ ← 클릭 시 상품 상세로 이동
└─────────────────────┘
```

### 반응형 브레이크포인트
| 화면 크기 | 카드 개수 | 용도 |
|-----------|-----------|------|
| 1280px 이상 | 5개 | PC |
| 1024~1279px | 4개 | 작은 데스크톱 |
| 768~1023px | 3개 | 태블릿 |
| 640~767px | 2개 | 큰 모바일 |
| 640px 이하 | 1.2개 | 모바일 (스와이프 힌트) |

### 스타일 특징
- 카드 모서리: `border-radius: 12px`
- 호버 효과: `translateY(-4px)`, 그림자 증가
- 이미지 줌: 호버 시 `scale(1.05)`
- 색상: 절제된 뉴트럴 톤 (회색/아이보리)
- 폰트: 시스템 폰트 스택

## 🔍 문제 해결

### Q. 빈 상태 메시지가 표시되지 않습니다.
**A.** 다음을 확인하세요:
1. 실제 리뷰가 0개인지 확인 (관리자 > 게시판 > 상품후기)
2. 브라우저 콘솔에서 JavaScript 오류 확인
3. CSS 파일이 정상적으로 로드되는지 확인

### Q. 데모 모드를 활성화했는데 빈 상태만 보입니다.
**A.** 다음을 확인하세요:
1. 페이지를 새로고침했는지 확인
2. 콘솔에서 `window.ATELIER_DEMO_MODE` 값 확인
3. `atelier-review-demo-data.js` 파일 로드 확인 (Network 탭)
4. JavaScript 오류 확인 (Console 탭)

### Q. 슬라이더가 작동하지 않습니다.
**A.** 다음을 확인하세요:
1. Swiper 라이브러리 로드 확인
2. 리뷰 개수가 3개 이상인지 확인
3. 콘솔에서 에러 메시지 확인

### Q. 데모 이미지가 표시되지 않습니다.
**A.** 다음을 확인하세요:
1. 이미지 파일이 서버에 업로드되었는지 확인
2. 이미지 경로가 정확한지 확인
3. Fallback 이미지가 표시되는지 확인
