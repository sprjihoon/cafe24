- **우리 스킨**: 브랜드 + 범용 = 하이브리드

## 기술 스택

### 라이브러리
- **Swiper.js** - 슬라이더 (히어로, WEEKLY BEST, SHORTS, 포토리뷰)
- **Intersection Observer API** - 스크롤 효과, 사이드 네비
- **YouTube IFrame API** - 영상 임베드
- **Instagram Basic Display API** - SNS 피드 (선택)

### Cafe24 모듈
- `product_listmain_1, 2, 3` - 메인 상품 진열
- `product_listnormal` - 일반 상품 목록
- `Layout_category` - 네비게이션
- `board` - 포토리뷰
