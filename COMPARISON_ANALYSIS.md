## 5. 우리 스킨에 부족한 요소 (실제 비교 기준)

### 🔴 우선순위 높음 - 헤더/네비게이션

1. **이미지 로고 지원**
   - 현재: 텍스트 + JavaScript로 교체
   - 개선: 관리자에서 이미지 업로드 가능하도록

2. **우측 유틸리티 바 확장**
   ```
   추가 필요:
   - 회원가입 버튼 (+포인트 혜택 표시)
   - 커뮤니티 링크
   - 언어 변경 (다국어 쇼핑몰용)
   ```

3. **햄버거 메뉴**
   - 모바일에만 있고 PC에는 없음
   - PC에도 사이드 메뉴 필요 시 추가

4. **메뉴 확장**
   ```
   추가 검토:
   - BEST 50 (인기 상품)
   - ON SALE (세일 섹션)
   - 무관베네 (관심 상품 빠른 접근)
   ```

### 🔴 우선순위 높음 - 메인 배너

5. **히어로 배너 개선**
   ```
   현재: 420px 고정 높이
   개선: 전체 화면 (100vh) 옵션 추가
   
   현재: 이미지만 지원
   개선: 영상 배경 지원 (HTML5 video 또는 YouTube)
   ```

### 🔴 우선순위 높음 - 관리 시스템
1. **config.xml + settings.html** (관리 UI)
   - 마키 배너 텍스트 설정
   - 히어로 슬라이더 ON/OFF
   - 브랜드 색상 선택기
   - 카테고리 카피 편집

2. **Layout_category 모듈 적용**
   - 현재 네비게이션 cate_no 하드코딩 문제 해결
   - 관리자에서 카테고리 추가/수정 시 자동 반영

3. **실용적 정보 섹션**
   - 무료배송 안내
   - 결제 혜택 (무이자 할부 등)
   - 고객 서비스 정보

### 우선순위 🟡 중간
4. **푸터 디자인 개선**
   - 현재: 기본 Cafe24 푸터 + 커스텀 뉴스레터
   - 목표: 세련된 브랜드 푸터 (링크 정리, 레이아웃)

5. **서브페이지 완성도**
   - 장바구니 스타일
   - 주문/결제 페이지
   - 마이페이지
   - 게시판

6. **반응형 완성도**
   - 태블릿 최적화 (현재 PC/모바일만)
   - 다양한 화면 크기 테스트

### 우선순위 🟢 나중
7. **추가 기능 (선택)**
   - 이벤트/프로모션 섹션
   - 리크루트/입점 제휴 안내
   - 리뷰/갤러리 섹션

## 7. 결론 및 제안

### 현재 상황
- **디자인센터 LE BLANC**: 일반 쇼핑몰용, 관리 편의성 중심
- **우리 스킨**: 브랜드 쇼핑몰용, 디자인 & 무드 중심

### 방향성 제안

#### 옵션 A: 브랜드 스킨으로 특화 (추천)
```
현재 강점(브랜드 무드, 커스텀 슬라이더)을 살리면서
최소한의 관리 시스템만 추가
→ "브랜드 중심 럭셔리 스킨"으로 차별화
```

**추가 작업:**
- config.xml (마키 배너, 색상, 카피 편집)
- Layout_category 적용
- 무료배송/고객서비스 정보 섹션
- 푸터 개선

#### 옵션 B: 디자인센터 스킨처럼 범용화
```
관리 UI 대폭 확장, 세일/이벤트 섹션 추가
→ "어떤 쇼핑몰에도 적용 가능한 스킨"
```

**추가 작업:**
- 옵션 A의 모든 작업
- EVENT/RECRUIT 섹션 템플릿
- 할인/프로모션 배너 시스템
